README

kernel.asm		Source Code for Assembly Code.
kernel.c		Source Code for C.
kasm.o			Compile Assembly Code.
kc.o			Compile C Code.
link.ld			Linker for linking kasm.o and kc.o
build.sh		Bash Code for terminal automation to run everything in compile/run setup.
kernel.bin 		Full build linked together and ready to run, but it is not an ISO.
ISO-Convertion.iso	The final finel converted from kernel.bin into an ISO to be fully boot from.

The ISO-Convertion folder is the strcuture that it is need it to produces an ISO.
